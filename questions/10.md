# Multiple Choice Question

What is the output of the following program?

```java
public class Main {
    public static void main(String[] args) {
        int firstNumber = 6;
        int secondNumber = 3;
        
        firstNumber += secondNumber;
        System.out.println(firstNumber);
        firstNumber /= secondNumber;
        System.out.println(firstNumber);
    }
}
```

## Correct

```text
9
3
```

## Incorrect

```text
6
6
```

## Incorrect

```text
9
2
```
